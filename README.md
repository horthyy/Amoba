# Amőba

Klasszikus zártpályás amőba játék. Kétszemélyes játék, lehet gépi játékos ellen is játszani. Az nyer, aki először kirak 5 egyformát egymás mellé a saját jeléből, sorba, oszlopba vagy diagonálisan. A pálya 15x15 ... 30x30 méret között állítható.

(A program a Bevezetés a programozásba II. tantárgy keretében készült harmadik beadandó munka.)
